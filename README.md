<h1 align="center">Codificador & Decodificador🗒️</h1>
<p align="center">Se consiste em um script que codifica e decodifica strings.</p>

<h2>O script faz:</h2>

<h3>Codificação⚙️</h3>

**Substitui as letras:**

*"a, e, i, o, u"*

**para:**

*"i, nter, mes, ber, fat"*

**respectivamente.**

<h3>Decodificação⚙️</h3>

**Substitui as letras:**

*"i, nter, mes, ber, fat"*

**para:**

*"a, e, i, o, u"*

**respectivamente.**

`◽JavaScript`

`◽HTML`

`◽CSS`

`◽Visual Studio Code`
